# Introduction
 
Welcome to the documentation for the `Kuts.club` project. This documentation is intended to provide a comprehensive guide to the API and its endpoints. If you have any questions or need further assistance, please don't hesitate to reach out to us.
## What is Kuts.club?

Kuts.club is a project that aims to provide a simple and easy-to-use URL shortening service. The API allows users to create short links for long URLs, as well as retrieve information about existing links. The project is open-source and free to use, and we welcome contributions from the community.
## Documentation

The documentation is available at [https://docs.kuts.club](https://docs.kuts.club).
